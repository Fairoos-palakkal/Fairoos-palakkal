## How I Work

I approach AI engineering as a full-stack discipline: the model is only one part of the system. My focus is on **the complete loop** — data ingestion, preprocessing, model training, evaluation against a measurable baseline, and deployment behind a production-grade API.

Every project I ship answers three questions: **What problem does it solve? What does the output look like? How do you measure whether it worked?**

If a model can't answer those questions, it's not ready.
